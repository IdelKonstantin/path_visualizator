1) Установить python3:

	sudo su
	apt install python3

2) Проверить версию (должна быть не ниже 3.5):

	python3 --version


3) Дать скрипту визуализации права на выполнение: 

	chmod +x ./viz.py

4) Запускать так:

	<ваша программа формирующая выходные данные> | ./viz.py

	Например, так: 

	./test | ./viz.py

5) Настроить отображение можно в файле viz.py в строках 16 - 20, здесь:

	POLYGON_X_SIZE - размер полигона по Х
	POLYGON_Y_SIZE - размер полигона по У
	FONT_SIZE      - размер шрифта для подписи в узле
	FONT_GAP       - зазор между подписью узла по вертикали
	NODE_RADIUS    - размер окружности в узле
	ROUND_DIGITS   - знак, до которого округляется время и курсовой угол
	SHOW_ANNOTATIONS - показывать подписи в узлах (True / False)
